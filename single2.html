The Process

When making a road sign classifier, there are 3 main steps to go through. After a sample image goes through these three steps, the computer should be able to identify what type of sign is in the image.

Preprocessing

The first step is preprocessing. Computers have a hard time analyzing RGB values. So the image is converted to greyscale, where there is only one value per pixel. This steps goal is so that when the image is done preprocessing, the computer can actually understand the image.

Convolutional Neural Network

Now we have to make it so that the computer can actually determine what sign it actually is. It does this by using a CNN to point out key features of the image. The more layers there are, the more features the network looks, for thus being more accurate. Now it just needs to output it by assigning it to a class. Each one of these classes are assigned to a specific sign. This completes our process of identifying road signs.

The Output

The computer now has a really good idea of what image the sign is by using preprocessing to understand it, and a CNN to actually identify it.
